Your project is empty and therefore quite sad. Here are some things to do:

Graphics are organized into groups and objects. An object is the actual graphic, the group is just to make them easier to find.

Press this button to create a new group and object and start drawing stuff.

Starting from scratch is hard and doing hard stuff is exhausting. Try loading an example project to learn how Antrax works.

Or maybe just look at some pretty pictures.

There are many shortcuts to make things easier and more efficient. Press the keyboard button to view them.

While you can use Antrax for generic pixel art, it is optimized to make things easier when creating graphics for the Atari 7800. Which can be quite confusing. Press the help button for more information about why things are the way they are.
